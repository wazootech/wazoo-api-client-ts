# Agent guidelines

## What this repo is

This repository contains the TypeScript client for the Wazoo API.

## How to work here

- Keep generated and hand-written client code boundaries clear.
- Use `package.json` scripts for build, typecheck, and tests.
- Run `npm run typecheck` and `npm test` for behavior changes when practical.
- Update README examples when public client methods, options, or responses
  change.
